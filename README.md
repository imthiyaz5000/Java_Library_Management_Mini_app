A simple Java console project that manages books, users, and borrowing/return workflows.
Built using Core Java, OOP concepts, Collections, and clean modular structure.

Features

Add new books to the library

Search books by title

Borrow and return books

Track issued books

Clean OOP design using multiple classes

How to Run
1. Open the project in any Java IDE (Eclipse/IntelliJ).
2. Navigate to src/com/Library/app/Main.java
3. Run the Main class.
